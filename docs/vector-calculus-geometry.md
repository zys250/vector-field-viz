# 向量分析公式的几何解释

这份说明服务于项目内的交互章节，重点不是推导每一步，而是说明每个公式在画布上对应什么几何动作。

## 通量

通量看的是向量场穿过一条有向曲线或一个闭合边界的多少。

```text
Φ = ∫C F · n ds
```

如果 `F · n` 为正，表示沿法向穿出；为负，表示穿入。开放曲线的通量依赖曲线方向，因为方向决定法向；闭合曲线则常用外法向描述净流出。

## 环量

环量看的是向量场沿闭合回路切向推动一圈的总量。

```text
Γ = ∮C F · dr
```

环量适用于任意向量场，不只适用于旋转场。若回路上正向推动和反向推动抵消，总环量可以接近 0；若区域内旋度主要同号，总环量通常明显非零。

## 散度与高斯定理

散度是“单位面积/体积附近的净流出率”。在二维画布上，小圆探针测的是：

```text
局部通量 ≈ div F · 小圆面积
```

把一个区域切成许多小格子，每个小格子都有自己的净流出。相邻格子共享的内部边界会出现一边流出、另一边流入，所以内部贡献抵消；剩下的就是最外圈边界的净流出。

二维形式：

```text
∮∂S F · n ds = ∬S div F dA
```

三维形式：

```text
∯∂V F · n dS = ∭V div F dV
```

这就是“表面通量”和“体积散度”的关系：边界上看到的净流出，等于内部所有源汇强度的总和。

## 旋度与 Stokes 定理

二维旋度描述一点附近是否会拧动一个小桨轮：

```text
curl F = ∂Fy/∂x - ∂Fx/∂y
```

剪切流是很好的反例：它不一定绕某个中心旋转，但上下速度不同，仍然会让小桨轮自转。源流和常量场可以有明显速度，但没有局部自转时 curl 接近 0。

二维局部 Stokes 关系：

```text
∮∂S F · dr ≈ curl F · Area(S)
```

三维 Stokes 定理：

```text
∮∂S F · dr = ∬S (∇ × F) · n dS
```

直觉上，边界一圈的总切向推动，来自曲面内部每一点局部旋转的累加。

## Green 定理在本项目中的位置

二维高斯定理和二维 Stokes/环量关系都可以看作 Green 定理的两个侧面：

```text
∮∂S P dx + Q dy = ∬S (∂Q/∂x - ∂P/∂y) dA
```

当边界积分取切向分量时，它连接环量与旋度；当边界积分取法向分量时，它连接通量与散度。

## 二维电场章节的特别说明

项目中的二维电场使用 `E = kQ/r · r_hat`，目的是让闭合曲线通量只由包围净电荷决定：

```text
∮C E · n ds = 2πkQenc
```

真实三维静电学使用闭合曲面与 `1/r²` 衰减。本项目的二维模型是教学类比，不应直接当成三维库仑场。
