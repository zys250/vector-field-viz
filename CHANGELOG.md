# Changelog

## 0.1.0 (2026-06-08)

### Added
- 10 个向量场交互实验：入门、通量、环量、散度、旋度、电场、磁场、水流、气体、3D 投影
- React + Canvas 2D 渲染引擎，支持场线、箭头、热力图、示踪粒子
- 运行质量档位（high / balanced / safe），适配不同性能设备
- Windows 桌面应用（Electron portable 包）
- Android 支持（Capacitor）
- HarmonyOS 移动端壳工程
- PWA / Service Worker 离线支持
- 核心数学回归测试（散度定理、旋度、剪切流等）
- CI（GitHub Actions：test + lint + build）
