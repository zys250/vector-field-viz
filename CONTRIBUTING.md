# Contributing

感谢你愿意维护 Vector Field Lab。这个项目优先服务教学清晰度和长时间演示稳定性，修改时请尽量保持“概念准确、交互直观、低性能设备可跑”这三条线。

## 本地开发

```bash
npm install
npm run dev
```

提交前请运行：

```bash
npm run check
```

## 维护原则

- 不为了视觉效果牺牲概念准确性。旋度、散度、通量、环量的说明要区分局部量、积分量和边界方向。
- 继续使用 React + Canvas 2D 架构，除非有明确性能证据需要更换渲染方案。
- 粒子、场线、热图等昂贵绘制应接入 `renderBudget`，避免硬编码数量。
- 长时间运行要优先稳定：动画步进使用 `dt`，页面隐藏后暂停/降载，恢复时重置时间基准。
- 新增中文说明时保持 UTF-8，避免出现乱码、替换字符或半截公式。

## 推荐验收

1. 首页能打开，章节导航正常。
2. Ch3 环量、Ch4 散度、Ch5 旋度、Ch6 高斯圈读数正常。
3. Ch5 切到“剪切流”时，小桨轮应能明显转动；切到“源流/常量场/鞍点场”时不应自转。
4. 在“运行设置”里切换 high/balanced/safe，粒子数、场线数量和热图密度应随预算变化。
5. 运行 `npm run check`。

## 发布前注意

公开到 GitHub 前请确认开源协议、应用图标、release 包命名，以及 README 中的安装/运行说明是否与当前脚本一致。
